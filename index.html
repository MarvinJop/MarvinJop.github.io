HTML tags

Tag Tag Name Description
Document Ouline
<!DOCTYPE> Document Type Defines the document type. Example: <!DOCTYPE html>
<html> Html Represents the root of an HTML document
<head> Head Contains metadata for the document/page information
<body> Body Defines the body of the document/page content
Text Markup
<strong> Strong Used to define text with strong importance, the content inside this tag is typically displayed in bold
<em> Emphasis Defines emphasized text. The content inside is typically displayed in italic
<sub> Subcript Defines subscript text. Appears half a character below the normal line, and is sometimes rendered in a smaller font. Can be used for chemical formulas, like H2O.
<sub> Superscript Defines superscript text. Appears half a character above the normal line, and is sometimes rendered in a smaller font. Can be used for formulas, like X2 or footnotes. Example to show how <sup> is rendered[1]
Document Structure
<h[1-6]> Heading Used to define HTML Headings. <h1> for main title, <h2> for subtitle and so on.
<div> Div Defines a division or section in an HTML document. It is used as a container for HTML elements.
<span> Span Is and inline container, used to mark up a part of a text or a document. is easily styled with CSS or manipulated with JavaScript.
<br> Break Insert a single line break. Is and empty tag which means that it has no ending tag.
Tables
<table> Table Defines a table.
<thead> Table Head Groups the header content of a table.
<tbody> Table Body Groups the body content of a table.
<tr> Table Row Defines a row in a table.
<td> Table Data Defines table data.
<tfoot> Table Footer Groups the footer information of a table.

CSS Selectors

Basic Selectors Description Syntax
<Universal [*]> Selects all elements on the page * {property: value;}
Type Selects all HTML tag/element of given type.
  • p
  • h[1-6]
  • a
  • img
 element {property: value;}
Id Selects and element based on the value of its unique id attribute #elementId {property: value;}
class Selects all elements that have the given class attribute .class {property: value;}
